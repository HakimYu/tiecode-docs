# 文件操作教程
 
文件操作具有以下方法

```
移动文件(原路径 为 文本型,新路径 为 文本型)
重命名文件(原名称 为 文本型,新名称 为 文本型) 为 逻辑型
取文件前缀名(路径 为 文本型) 为 文本型
取文件后缀名(路径 为 文本型) 为 文本型
取文件名(路径 为 文本型) 为 文本型
取文件MD5(路径 为 文本型) 为 文本型
取文件SHA1(路径 为 文本型) 为 文本型
取文件CRC32(路径 为 文本型) 为 文本型
追加文本(路径 为 文本型,文本 为 文本型)
删除文件(路径 为 文本型) 为 逻辑型
创建目录(路径 为 文本型) 为 逻辑型
删除目录(路径 为 文本型) 为 逻辑型
是否为目录(路径 为 文本型) 为 逻辑型
是否为隐藏文件(路径 为 文本型) 为 逻辑型
文件是否存在(路径 为 文本型) 为 逻辑型
取文件编码(路径 为 文本型) 为 文本型
读入文本文件(文件名 为 文本型) 为 文本型
写出文本文件(文件名 为 文本型,内容 为 文本型)
读入字节文件(路径 为 文本型) 为 字节[]
写出字节文件(文本型,字节[]) 为 逻辑型
取文件尺寸(路径 为 文本型) 为 长整数型
读入资源文件(文件名 为 文本型,编码 为 文本型) 为 文本型
读入资源文件(文件名 为 文本型) 为 文本型
写出资源文件(文件名 为 文本型,输出路径 为 文本型) 为 逻辑型
寻找文件关键词(文件名 为 文本型,关键词 为 文本型) 为 文本型
复制文件(路径 为 文本型,新路径 为 文本型) 为 逻辑型
创建文件(路径 为 文本型) 为 逻辑型
打开文本文件_读(路径 为 文本型,编码 为 文本型) 为 逻辑型
关闭读() 为 逻辑型
读一行() 为 文本型
打开文本文件_写(路径 为 文本型,编码 为 文本型) 为 逻辑型
关闭写() 为 逻辑型
写一行(内容 为 文本型) 为 逻辑型
取子目录(路径 为 文本型) 为 文本型
取子文件列表(目录 为 文本型) 为 文本型[]
取子文件集合(目录 为 文本型) 为 集合
取文件修改时间(路径 为 文本型) 为 文本型
打开文件(路径 为 文本型)
```

我们可以使用
`文件操作.方法名`来使用对应方法


如复制文件使用

```
文件操作.复制文件(存储卡路径 + "/1.jpg",存储卡路径 + "/test/1.jpg")
```

将存储卡路径下的`1.jpg`复制到存储卡路径下的`test文件夹内的1.jpg`
